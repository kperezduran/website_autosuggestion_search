Odoo 12 Autocomplete product search in website

Autocomplete product search in website

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Features

  • Allow user to quickly add products in to cart by uploading csv,xls,xlsx file or set comma separated value in text area.

How to use

  • User can just type product name and related products will be listed in dropdown with image.


  • Click product image in search result to view full size image
